# 📈 라오어 VR 리밸런싱 매니저 (2주 주기)

이 앱은 라오어의 **밸류 리밸런싱(Value Rebalancing)** 전략을 기반으로, 기존 보유 주식을 포함하여 **2주(거래일 10일) 주기**로 포트폴리오를 관리할 수 있도록 설계되었습니다.

## 🚀 바로가기
- **URL:** `https://[본인아이디].github.io/[저장소이름]/`

## ✨ 주요 기능
- **초기 밸류($V_0$) 반영:** 기존에 보유한 주식의 가치(수량 × 주가)를 시작점으로 설정 가능.
- **G값 커스터마이징:** 2주마다 증가할 목표 가치(G)를 자유롭게 조정.
- **2주 주기 가이드:** 기록을 확정할 때마다 1사이클이 증가하며 차기 목표 밸류(TV) 자동 산출.
- **실시간 시세 반영:** Yahoo Finance API를 통한 실시간 가격 조회.
- **PWA 지원:** 모바일 브라우저에서 '홈 화면에 추가' 시 앱처럼 설치 가능.

## 🛠 사용 방법
1. **종목 추가:** 티커(TQQQ 등)를 입력하고 초기 밸류($V_0$)와 G값을 설정합니다.
2. **리밸런싱:** 2주에 한 번 앱을 켜서 가이드에 나온 금액만큼 매수/매도합니다.
3. **기록 확정:** 실제 매매한 가격과 수량을 입력하고 '기록 확정'을 누르면 다음 사이클로 넘어갑니다.
4. **백업:** 브라우저 캐시 삭제 시 데이터가 소실될 수 있으니 정기적으로 '백업'하세요.
