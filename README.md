# 📈 VR 리밸런싱 매니저 (VR Manager)

라오어의 **밸류 리밸런싱(Value Rebalancing)** 전략을 웹에서 간편하게 관리하기 위한 도구입니다.

## 🚀 서비스 주소
- **URL:** `https://[본인의아이디].github.io/[저장소이름]/`

## ✨ 주요 기능
- **G값 커스터마이징:** 회차별 목표 밸류 증가분(G) 수동 설정 및 실시간 계산.
- **실시간 시세 반영:** Yahoo Finance API 기반 실시간 가격 업데이트.
- **PWA 지원:** 모바일 및 데스크톱에서 '앱'으로 설치하여 사용 가능.
- **데이터 보안:** 서버 저장 없이 브라우저 로컬 저장소(`localStorage`)에만 데이터 보관.
- **백업/복원:** JSON 파일 형태의 데이터 내보내기 및 가져오기 기능.

## 🛠 설치 및 사용법
1. GitHub 저장소를 팔로우하거나 파일을 다운로드합니다.
2. `index.html`을 브라우저로 엽니다. (또는 GitHub Pages로 배포)
3. 모바일 브라우저(Chrome, Safari)에서 **'홈 화면에 추가'**를 누르면 앱처럼 설치됩니다.

## ⚠️ 주의사항
- 브라우저 쿠키나 캐시를 삭제하면 로컬 데이터가 지워질 수 있습니다.
- **정기적으로 '백업' 버튼을 눌러 JSON 파일을 저장하는 것을 권장합니다.**
